# Wares
A C/C++ package manager compatible with all your favorite build tools.

## Features:
- Updater script
- Premake5!
- CMake!
- package.lock
- global package cache!

### Consume packages from:
- Github
- Gitlab
- Bitbucket
- Private repositories
- URLs
- Binary packages
- pkg-config

### Versioning!
- By release, tag, or commit
